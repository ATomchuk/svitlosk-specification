# Purpose

Clarify misleading wording about root-level document disposition.

---

# 1. Original Wording (Misleading)

> "~198 non-Foundation documents violate the Principle — all scheduled for archive"

This implied ALL 198 files go to archive/. This is incorrect.

---

# 2. Corrected Wording

> "~198 non-Foundation root-level documents are scheduled for relocation according to the approved Repository Blueprint."

---

# 3. Relocation vs Archiving

| Term | Definition | Scope |
|------|-----------|-------|
| Relocation | Moving a document to its correct architectural destination | ALL 198 files |
| Archiving | Moving historical materials into archive/ | SUBSET of 198 files |

Relocation includes archiving but is NOT limited to archiving.
