# Purpose

This document defines the semantic boundaries for every Editorial Principle, establishing what each principle owns and what it does NOT own.

---

# EP-001: Reader First Principle

**Owns:**
- Audience priority decisions
- Community benefit assessment
- Reader needs evaluation

**Does NOT own:**
- Technical delivery mechanisms
- Data source selection
- Rendering format choices
- Lifecycle state management
- Platform configuration
