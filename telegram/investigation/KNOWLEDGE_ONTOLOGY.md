## Models of Knowledge

### Model 1: Knowledge as True Belief

**Definition:** Knowledge is justified true belief about Reality.

**Evidence:**

- Knowledge requires a Knower
- Knowledge is about Reality
- Knowledge can be true or false (belief)

**Fit:** GOOD — classical epistemological definition.

---

### Model 2: Knowledge as Understanding

**Definition:** Knowledge is understanding gained from Observation.

**Evidence:**

- Observation produces Knowledge
- Knowledge is what Knower understands
- Knowledge can be transferred

**Fit:** GOOD — explains Knowledge formation.

---

### Model 3: Knowledge as Mental State

**Definition:** Knowledge is a state of the Knower's mind.

**Evidence:**

- Knowledge exists in minds
- Knowledge can be forgotten
- Knowledge is subjective

**Fit:** PARTIAL — explains Knowledge location but not content.

---

### Model 4: Knowledge as Information Processed

**Definition:** Knowledge is Information that has been processed by a Knower.

**Evidence:**

- Information becomes Knowledge when understood
- Knowledge is processed Information
- Knowledge is Information + Understanding

**Fit:** GOOD — explains relationship between Information and Knowledge.

## Knowledge Ontology Summary

| Property | Value |
|----------|-------|
| Exists independently? | NO — requires knower |
| Has identity? | NO — Knowledge is state, not thing |
| Has owner? | YES — knower |
| Can change? | YES — Knowledge can grow, shrink, change |
| Can be copied? | PARTIALLY — can be transferred, but each copy is separate |
| Can be represented? | YES — by Information, Documents |
| Can contain other concepts? | YES — Knowledge can contain Facts, Information |
| Can exist without humans? | NO — requires knower |
| Must be persistent? | NO — Knowledge can be forgotten |

# Key Insight

**Knowledge is what the Knower HOLDS about Reality.**

**Knowledge requires a Knower to exist.**

**Knowledge can be true or false, can change, can be forgotten.**

**Knowledge comes BEFORE Information in the chain.**
