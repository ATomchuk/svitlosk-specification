# Reconstructed Ontology

## What "Object" Means in This Repository

**The repository does NOT explicitly define "Object".**

**However, the repository implicitly uses "Object" to mean:**

A thing that:

1. Has existence (can be pointed to)
2. Has identity (can be distinguished from other things)
3. Has properties (can be described)
4. Has relationships (can be connected to other things)
5. Has lifecycle (can be created, changed, destroyed)
