# Chain Analysis

## Where Publication First Appears

**Between Interpretation Result and Representation.**

**Evidence:**

- TJS-021 §4.1: "Generated — The state in which a Publication has been created from normalized data but has not yet been published to Telegram"
- Publication is created from Interpretation Result
- Publication is rendered as Representation

---

## Where Communication Occurs

**After Carrier, before Recipient.**

**Evidence:**

- Communication is the transfer process
- Communication happens during delivery
- Communication creates Understanding

---

# Key Insight

**The chain has 15 layers.**

**Publication is layer 9 — the INFORMATION OBJECT created for Communication.**

**Communication is layer 13 — the PROCESS that transfers meaning.**

**Publication and Communication are DIFFERENT layers.**
