## What Is Representation?

**Definition:** Representation is a portrayal of Information in a specific format or medium.

**Evidence:**

- Information can be represented as text, image, audio
- Representation is the FORM, not the CONTENT
- Multiple Representations can convey same Information

---

## Does Representation Contain Information?

**Answer:** YES

**Evidence:**

- Telegram Message contains Publication content
- Database row contains data
- Markdown contains formatted text

**Conclusion:** Representation CONTAINS Information.

---

## Does Representation Merely Reference Information?

**Answer:** NO

**Evidence:**

- Representation contains the actual content
- Representation is not just a pointer
- Representation carries the Information itself

**Conclusion:** Representation CONTAINS, not merely references.

---

## Can Multiple Representations Describe One Information?

**Answer:** YES

**Evidence:**

- Same outage can be text, image, or schedule
- Same data can be JSON, XML, or CSV
- Same Publication can be Telegram Message or Repository Object

**Conclusion:** Multiple Representations CAN describe same Information.
