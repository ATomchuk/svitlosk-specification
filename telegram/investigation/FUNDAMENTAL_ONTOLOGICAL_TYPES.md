# Fundamental Ontological Types

**Definition:** Physical existence independent of observers.

**Why it exists:** Foundation of everything — what the system models.

**Properties:**

- Exists independently: YES
- Has identity: NO

- Can change: YES
- Can disappear: NO
- Can contain: YES
