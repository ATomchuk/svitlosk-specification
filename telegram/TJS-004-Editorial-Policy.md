# TJS-004 — Editorial Policy

Status: Approved (Затверджено)

Specification ID: TJS-004

Component: Editorial Policy

Author: SvitloSk Project

---

# Purpose

This document defines the editorial standards for all text publications in the SvitloSk Telegram Journal.

The objective is to ensure that every publication is consistent, easy to read, predictable and automatically generated from structured data.

---

# Editorial Principles

The Telegram Journal follows these principles:

- Territory-first presentation.
- One post — one territory.
- Maximum readability.
- Minimal formatting.
- Consistent structure.
- No decorative elements.
- Automatic generation without manual editing.

---

# Territory

A post represents only one territory.

Possible territories include:

- STAROKOSTIANTYNIV
- SAMCHYKY SO
- VELYKI MATSEVYCHI SO
- ...

Territory name is always the first element of the publication.

It is written using uppercase letters.

Example:

STAROKOSTIANTYNIV

SAMCHYKY SO

---

# Publication Package

The daily publication package consists of:

- one post for Starokostiantyniv;
- one post for every starosta district with available information;
- separate posts containing information for the following day.

Posts without information are not created.

---

# Message Categories

Inside every post information is presented in the following order:

1. Planned outages.
2. Emergency outages.

If one category has no records, it is omitted.

---

# Category Titles

Category titles use emoji indicators.

Examples:

🟠 Planned outages

🔴 Emergency outages

Category titles are displayed in bold.

---

# Publications For Tomorrow

Information for the following day follows the same structure.

Heading example:

🟠 Planned outages

for 09 July (tomorrow)

These publications are temporary and may be removed after the reporting day.

---

# Time Format

Time is displayed before every outage interval.

Example:

🕘 09:00–17:00

Parentheses are not used.

---

# Settlements

Settlement names are displayed in bold.

Administrative prefixes are omitted.

Correct:

**Velyki Matsevychi**

Incorrect:

**v. Velyki Matsevychi**

---

# Addresses

Official address abbreviations are preserved.

Examples:

вул.

пров.

площа

просп.

Addresses are displayed line by line.

Example:

вул. Козацька 1, 4, 5

вул. Лісова 1, 2, 3

пров. Садовий 1, 3, 8

---

# Formatting

Only the following formatting is permitted:

- bold;
- plain text.

Italic, underline, spoiler, blockquote, code blocks and other formatting are prohibited.

---

# Update Information

Each publication ends with the update timestamp.

Example:

Updated: 17:34

---

# Branding

Posts do not contain signatures.

Posts do not contain decorative separators.

Posts do not contain location icons.

The Telegram channel itself represents the SvitloSk identity.

---

# Terminology

The project uses the term:

Outage

The terminology must remain consistent throughout every publication.

---

# Automatic Generation

Every publication is generated automatically from structured data.

Manual editing is not part of the standard publication workflow.

---

# Compliance

Every publication generated by SvitloSk shall comply with this editorial policy.
