# 2. Recommendation

1. **Standard IT term** — "Движок" is the universally accepted Ukrainian translation for "Engine" in software contexts
2. **Microsoft/Google/ISO aligned** — major tech companies use "Движок" for "Engine"
3. **Preserves proper name** — "Движок публікацій" preserves the functional identity of the Component
4. **Natural Ukrainian** — immediately understood by Ukrainian developers
5. **Already in Glossary** — established with 149 occurrences
6. **Low ambiguity risk** — no conflicting translations exist
