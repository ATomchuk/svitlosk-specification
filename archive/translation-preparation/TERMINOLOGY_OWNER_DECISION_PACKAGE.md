# 1. Publication Engine

- **English Term:** Publication Engine
- **Meaning inside SvitloSk:** The architectural Component implementing the Publisher Role. Responsible for generating and distributing Publications. One of 8 primary components.
- **Candidate translations:**
  1. Движок публікацій — Natural Ukrainian IT term, widely understood
  2. Механізм публікацій — More formal, less common in IT
  3. Система публікацій — Too broad, implies entire system
  4. Компонент публікацій — Architecturally accurate, loses "engine"
  5. Генератор публікацій — Captures "generates" but not "distributes"
- **Recommendation:** Движок публікацій — most natural, already used in Ukrainian IT for "engine" concepts
- **Decision:** **STATUS: PENDING OWNER APPROVAL**

# 2. Canonical Equality

- **English Term:** Canonical Equality
- **Meaning inside SvitloSk:** The guarantee that two publications generated from identical datasets SHALL be byte-for-byte identical. Equivalent publications SHALL NOT generate Telegram edits.
- **Candidate translations:**
  1. Канонічна рівність — Already in Glossary, direct translation
  2. Канонічна тотожність — Stronger identity semantics
  3. Канонічна ідентичність — Captures "identical output"
  4. Стандартна рівність — Loses "canonical" semantics
  5. Канонічна однаковість — Natural but less formal
- **Recommendation:** Канонічна рівність — already established in the Glossary
- **Decision:** **STATUS: PENDING OWNER APPROVAL**

# 3. Powered

- **English Term:** Powered
- **Meaning inside SvitloSk:** A graphic rendering shorthand meaning "NOT currently experiencing an outage." Used as a color label (Orange) and timeline state (opposite of "Outage"). NOT about electrical supply in general.
- **Candidate translations:**
  1. Без знеструмлення — Directly says "no outage" — matches project language
  2. Норма — Short, means "normal state"
  3. Електропостачання — Standard Ukrainian but misleading in this context
  4. Забезпечення — Incomplete without context
  5. Відсутність відключення — "Absence of disconnection" — long
- **Recommendation:** Без знеструмлення — matches the project's linguistic model centered on "знеструмлення"
- **Decision:** **STATUS: PENDING OWNER APPROVAL**
