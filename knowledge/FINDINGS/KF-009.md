# KF-009_Domain_Architecture_Separate

**Document ID:** KF-009

**Title:** Domain and Architecture Are Separate Kernels

**Status:** Validated

---

# Statement

Domain Kernel and Architecture Kernel have ZERO overlap. Publication is the ONLY concept in both kernels.

---

# Meaning

The repository contains two independent ontological kernels: Domain (what the system models) and Architecture (how the system is implemented). These kernels have no overlapping concepts except Publication, which bridges them.
