# KF-010_Publication_Is_Information_Object

**Document ID:** KF-010

**Title:** Publication Is Information Object for Communication

**Status:** Validated

---

# Statement

Publication is an Information Object created specifically for Communication purpose.

---

# Meaning

Publication is not pure Information (which has no identity or lifecycle) and not pure Communication (which is a process). Publication is an Information Object — Information that has been given identity, lifecycle, and purpose for Communication.

---
