# Research Ontology

## Fundamental Research Objects

| Object | Definition | Category |
|--------|------------|----------|
| Question | An inquiry requiring investigation | Process |
| Problem | A situation requiring solution | State |
| Hypothesis | A proposed explanation | Knowledge |
| Observation | A fact noticed during investigation | Knowledge |
| Evidence | Proof supporting or contradicting a claim | Knowledge |
| Finding | A validated conclusion from investigation | Knowledge |
| Knowledge Unit | An atomic piece of architectural knowledge | Knowledge |
| Model | A conceptual representation of reality | Representation |
| Contradiction | A conflict between two statements | Relationship |
| Decision | A choice made by the Architect | Process |
| Verification | Confirmation that a claim is supported | Process |
| Audit | Independent examination of claims | Process |
| Lock | Permanent closure of a concept | State |
| Principle | An architectural truth that applies across contexts | Knowledge |
| Ontology | The structure of what exists | Knowledge |
| Definition | A precise statement of meaning | Knowledge |
| Assumption | An unverified belief | Knowledge |
| Investigation | A systematic inquiry into a subject | Process |
| Research Session | A bounded period of investigation | Process |
| Architect | The decision-maker | Role |
| Repository | The permanent storage of knowledge | Collection |
| Evidence Register | Collection of all evidence | Collection |
| Model Inventory | Collection of all models | Collection |
| Contradiction Register | Collection of all contradictions | Collection |
| Knowledge Base | Collection of all knowledge | Collection |
