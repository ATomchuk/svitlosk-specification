# 2. Case Scope

## 2.1 What This Case Investigates

The ARCHITECTURAL LEVEL of every entity.

NOT documentation.

NOT repository structure.

ONLY architectural level validation.

## 2.2 What This Case Does NOT Investigate

- Documentation design
- Repository structure
- Implementation design
- Entity renaming
- Entity removal

## 2.3 Investigation Constraint

For EVERY known entity determine its architectural nature.

Every entity MUST belong to exactly one architectural level.

Verify the whole ontology.

Detect every place where different architectural levels are mixed.
