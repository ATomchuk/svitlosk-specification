# 2. Contradictions

## 2.1 CONTRADICTION-001: Publication Definition vs Queue Graph

### Repository Statement

> "Publication: A public information message generated by the SvitloSk system." (GLOSSARY §3)

**Implication:** Publication is a TEXT MESSAGE.

### This Investigation Finding

Queue Graph is an IMAGE, not a text message.

### Analysis

Queue Graph does not fit the definition of "Publication" as a text message.

### Classification

POTENTIAL CONTRADICTION — Queue Graph may not be a "Publication."

### Severity

MEDIUM

---

## 2.2 CONTRADICTION-002: Territory Organization vs Queue Graph

### Repository Statement

> "Journal publications are grouped by Starosta Districts." (Architect Intent)

**Implication:** ALL publications are territory-organized.

### This Investigation Finding

Queue Graph is NOT territory-organized.

It is a SINGLE product for all Territories.

### Analysis

Queue Graph does not follow the territory organization pattern.

### Classification

REFINEMENT — Queue Graph is an exception to the rule.

### Severity

LOW

---

## 2.3 CONTRADICTION-003: Channel Independence vs Queue Graph Format

### Repository Statement

> "Telegram and Facebook DO NOT own information." (Architect Intent)

**Implication:** Information is channel-independent.

### This Investigation Finding

Queue Graph has DIFFERENT formats for different channels.

### Analysis

The DATA is channel-independent, but the FORMAT is channel-specific.

### Classification

REFINEMENT — Data independence vs format dependence.

### Severity

LOW

---

## 2.4 CONTRADICTION-004: Temporary Products vs Permanent Archive

### Repository Statement

> "Archived datasets SHALL remain immutable." (DATA_MODEL.md)

**Implication:** All data is archived permanently.

### This Investigation Finding

Temporary products (Planned Tomorrow, Queue Graph) are NOT archived.

### Analysis

Temporary products disappear, not archived.

### Classification

REFINEMENT — Temporary products have different archival rules.

### Severity

LOW
