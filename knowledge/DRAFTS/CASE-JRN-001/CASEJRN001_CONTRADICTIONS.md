# 2. Contradiction Register

## 2.1 CONTRADICTION-001: Journal as Service vs Journal as Channel

### Statements in Conflict

**Statement A (CHARTER.md §1):**
> "SvitloSk is an information service that analyses, interprets, archives and distributes officially published open data related to power outages within the Starokostiantyniv Community."

**Statement B (CHARTER.md §10):**
> "The SvitloSk Public Information Journal is the primary public communication channel of the project."

### Analysis

Statement A defines SvitloSk as a SERVICE.

Statement B defines the Journal as a CHANNEL.

If SvitloSk is a service, and the Journal is a channel, then the Journal is a COMPONENT of the service, not the service itself.

### Severity: HIGH

This affects the fundamental ontological category of Journal.

### Status: UNRESOLVED

Requires ontological clarification.

## 2.2 CONTRADICTION-002: Journal as Channel vs Journal as Archive

### Statements in Conflict

**Statement A (CHARTER.md §10):**
> "The SvitloSk Public Information Journal is the primary public communication channel of the project."

**Statement B (CHARTER.md §10):**
> "It serves as a public archive of processed information, daily publications, analytical summaries and official project communications."

### Analysis

Statement A defines Journal as a CHANNEL (delivery mechanism).

Statement B defines Journal as an ARCHIVE (storage mechanism).

A channel delivers information. An archive stores information.

These are DIFFERENT FUNCTIONS.

### Severity: HIGH

This affects the fundamental function of Journal.

### Status: UNRESOLVED

Requires functional clarification.

---

## 2.3 CONTRADICTION-003: Publication as Object vs Publication as Message

### Statements in Conflict

**Statement A (GLOSSARY.md §3):**
> "Publication: A public information message generated by the SvitloSk system for distribution through one or more supported communication channels."

**Statement B (GLOSSARY.md §3):**
> "Every Publication SHALL be derived exclusively from an Interpretation Result."

### Analysis

Statement A defines Publication as a MESSAGE.

Statement B defines Publication as derived from an Interpretation Result.

A message is a TRANSIENT object (sent and consumed).

An Interpretation Result is a PERSISTENT object (stored and processed).

If Publication is derived from Interpretation Result, it inherits persistence.

### Severity: MEDIUM

This affects the lifecycle of Publication.

### Status: UNRESOLVED

Requires lifecycle clarification.

---

## 2.4 CONTRADICTION-004: Publication Package as Collection vs Publication Package as Object

### Statements in Conflict

**Statement A (GLOSSARY.md §3):**
> "Publication Package: The complete collection of Publications generated during one Reporting Period."

**Statement B (GLOSSARY.md §3):**
> "Every Publication Package SHALL remain internally consistent."

### Analysis

Statement A defines Publication Package as a COLLECTION.

Statement B defines Publication Package as having INTERNAL CONSISTENCY.

A collection is a GROUPING (no internal state).

An object with internal consistency has STATE.

### Severity: LOW

This is a terminology ambiguity, not a true contradiction.

### Status: UNRESOLVED

Requires terminology clarification.

## 2.5 CONTRADICTION-005: Issue as Temporal Instantiation vs Issue as Editorial Edition

### Statements in Conflict

**Statement A (TELEGRAM_PUBLICATION_LIFECYCLE.md):**
> "An Issue opens when the first Publication for a calendar day is generated."

**Statement B (TELEGRAM_PUBLICATION_LIFECYCLE.md):**
> "The Issue represents one editorial edition of the Journal for that day."

### Analysis

Statement A defines Issue as a TEMPORAL INSTANTIATION (opens when first Publication is generated).

Statement B defines Issue as an EDITORIAL EDITION (represents one edition).

A temporal instantiation is defined by TIME.

An editorial edition is defined by CONTENT.

### Severity: MEDIUM

This affects the identity of Issue.

### Status: UNRESOLVED

Requires identity clarification.

### Statements in Conflict

### Severity: HIGH

This affects the fundamental ontological category of Journal.

### Status: UNRESOLVED

## 2.8 CONTRADICTION-008: Territory as Identity Dimension vs Territory as Publication Unit

### Statements in Conflict

**Statement A (GLOSSARY.md §4):**
> "Territory: A publication unit representing either: the Administrative Centre; one Starosta District."

**Statement B (GLOSSARY.md §4):**
> "A Territory is a logical publication entity rather than an administrative classification."

### Analysis

Statement A defines Territory as a PUBLICATION UNIT.

Statement B defines Territory as a LOGICAL PUBLICATION ENTITY.

A publication unit is a CONTAINER for Publications.

A logical publication entity is a CONCEPT that identifies Publications.

### Severity: LOW

This is a terminology ambiguity, not a true contradiction.

### Status: UNRESOLVED

Requires terminology clarification.
