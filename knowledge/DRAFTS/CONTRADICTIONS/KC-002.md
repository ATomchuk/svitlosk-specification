# Description

Publication belongs to Issue (TJS-000 §4), but Issue opens when first Publication is generated (TJS-021 §8.1). This creates a circular dependency.

---

# Affected Concepts

- Issue
- Publication
- Journal
- Lifecycle

---

# Current Understanding

- Publication belongs to Issue (structural relationship)
- Issue opens when first Publication is generated (temporal trigger)
- These are different aspects: structural vs temporal
- But they create apparent circularity

---

# Possible Resolutions

1. Accept circularity as design feature
2. Redefine Issue as independent concept
3. Redefine Publication as independent concept
4. Introduce explicit ordering
