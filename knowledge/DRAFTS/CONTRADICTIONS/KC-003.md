# KC-003_Telegram_Message_ID_As_Identity

**Document ID:** KC-003

**Title:** Telegram Message ID as Identity vs External Reference

**Status:** Draft

---

# Origin

CASE-001C, CASE-001D, CASE-001

---

# Description

Legacy TJS-007 §10 states "Ownership SHALL be determined by the stored Telegram Message ID." But Telegram Message ID is assigned after Publication creation, is platform-specific, and can be released.

---

# Affected Concepts

- Identity
- Identifier
- Publication
- Telegram Message ID
- Ownership

---

# Current Understanding

- Telegram Message ID is an external reference, not intrinsic identity
- Publication has identity (Territory + Date + Template) before Telegram Message ID
- Telegram Message ID can be released when Publication is removed
- Identity persists; Identifier can change

---

# Possible Resolutions

1. Accept Telegram Message ID as external reference only
2. Redefine ownership to use internal identity
3. Create explicit distinction between Identity and Identifier
