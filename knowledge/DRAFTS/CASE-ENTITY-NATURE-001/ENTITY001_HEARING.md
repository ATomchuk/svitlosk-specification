# 1. Purpose

This document establishes the scope, context and research questions for CASE-ENTITY-NATURE-001: Entity Nature Investigation.

---

# 2. Case Scope

## 2.1 What This Case Investigates

The FUNDAMENTAL NATURE of every entity.

NOT architectural levels.

NOT documentation.

ONLY what each entity IS.

## 2.2 What This Case Does NOT Investigate

- Discovering new entities
- Classifying architectural levels
- Redesigning documentation
- Renaming concepts

## 2.3 Investigation Constraint

For every existing entity determine WHAT IT IS.

Then determine ENTITY DEPENDENCY.

---

# 3. Context

## 3.1 Previous Investigations

All previous CASE investigations provide entity inventory.

## 3.2 The Goal

For every entity answer:

1. Is this entity TYPE, INSTANCE, COLLECTION, PROCESS, ROLE, SERVICE, DOCUMENT, ARTIFACT, STATE, EVENT, or something else?

2. Can it exist by itself?

3. Can multiple instances exist?

4. Does it have identity?

5. Can it change over time?

6. Can it be versioned?

7. Can it own other entities?

8. Can another entity own it?

9. Can it exist without Publisher?

10. Can Publisher exist without it?

## 3.3 The Constraint

No architecture.

No recommendations.

No implementation.

Only evidence.
