# SvitloSk

![Status](https://img.shields.io/badge/status-v3.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

> Automated Open Data Information System for electricity outage information.

SvitloSk collects, interprets and publishes official electricity outage information for the Starokostiantyniv Territorial Community (Ukraine). The system transforms complex technical data from Khmelnytskyioblenergo into clear, structured public information via Telegram.

---

## What is this repository?

This repository contains the **official specification** of the SvitloSk project — its architecture, engineering principles, technical standards and documentation.

It is NOT a code repository. It is the normative reference for how SvitloSk is designed and documented.
